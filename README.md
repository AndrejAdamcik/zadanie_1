# Snake - Moderná Klasika

Toto je moderná verzia klasickej hry Snake, vytvorená ako jednostránková webová aplikácia (SPA).

## Vlastnosti

*   **Moderný Dizajn:** Použitie Tailwind CSS pre čistý a responzívny vzhľad.
*   **Gradienty a Animácie:** Jemné animácie a farebné prechody pre príjemný vizuálny zážitok.
*   **Responzivita:** Prispôsobené pre desktopové aj mobilné zariadenia.
*   **Jednoduchá Štruktúra:** Všetko v jednom `index.html` súbore pre jednoduché nasadenie.

## Technológie

*   HTML5
*   Tailwind CSS (cez CDN)
*   JavaScript (Vanilla)

## Ako spustiť

Stačí otvoriť súbor `index.html` v akomkoľvek modernom webovom prehliadači.
